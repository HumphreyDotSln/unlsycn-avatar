## Ligatures

You can use the **`font-variant-ligatures: discretionary-ligatures`** CSS property to apply the ligatures in the unlsycn font.

Example:

`<a style="font-famliy: unlsycn; font-variant: discretionary-ligatures;">unlsycn</a>`
